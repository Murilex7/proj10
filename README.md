
# 📌 Manipulação de Arrays e Objetos em JavaScript

Uma aplicação simples e didática desenvolvida para praticar e demonstrar os conceitos fundamentais de estruturas de dados em JavaScript, focando na criação de arrays, estruturação de objetos e acesso indexado a propriedades.

O projeto resolve o problema de organizar informações detalhadas de múltiplos elementos (neste caso, alunos) em uma única variável, permitindo a extração rápida de dados específicos através de índices.

---

## 🚀 Funcionalidades

* **Estruturação de Dados**: Criação de um array contendo 5 objetos complexos.
* **Modelagem de Entidades**: Cada objeto representa um aluno com propriedades de `nome`, `idade` e `materiaFavorita`.
* **Acesso Direto**: Demonstração prática de como acessar propriedades de elementos específicos utilizando a indexação do JavaScript (base zero).
* **Feedback Visual**: Exibição dos dados acessados na tela do usuário por meio de caixas de diálogo (`alert`).

---

## 🛠️ Tecnologias Utilizadas

* **JavaScript**: Linguagem de programação principal utilizada para a lógica e estrutura de dados.
* **HTML5**: Ambiente de execução básico para rodar o script no navegador (opcional).
